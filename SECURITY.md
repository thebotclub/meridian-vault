# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅ Yes    |
| older   | ❌ No     |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities via [GitHub Private Security Advisory](https://github.com/thebotclub/tribunal-vault/security/advisories/new).

### Our SLA

| Milestone | Target |
|-----------|--------|
| Acknowledgement | Within 48 hours |
| Fix / Patch | Within 14 days of confirmation |
| Public Disclosure | Coordinated with reporter after fix |

We appreciate responsible disclosure and will credit reporters (with permission).
